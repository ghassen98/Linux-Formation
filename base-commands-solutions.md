# Les commandes de base de Linux

## Exercice 1 : 

### Solution : ###


1. Déterminez qui vous êtes.

```
$ whoami
$ who am i
```

2. Déterminez qui est connecté.

```
$ users
$ who
$ w
```

3. Déterminez qui fait quoi.

```
$ id
```

```
$ date
```

2. Affichez le calendrier de l’année en cours.

```
$ cal 2019
```

```
$ cal 06 2023
```
